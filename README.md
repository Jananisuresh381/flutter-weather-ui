# Flutter Weather UI

A simple weather application UI designed using Flutter. This project focuses on layout design and UI components.

## Features
- Weather UI layout
- Temperature and weather condition display
- Clean and modern interface

## Technologies Used
- Flutter
- Dart

## Purpose
This project was created for Flutter UI practice and to improve widget structuring skills.
